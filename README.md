# inha-internet-programming-quiz

인하대 인터넷프로그래밍 시험 대비용 정적 퀴즈 페이지입니다. 첫 화면에서는 중간고사/기말고사 개념 퀴즈만 선택할 수 있게 구성했습니다.

## 방향

- 중간고사와 기말고사를 별도 페이지로 분리합니다.
- 문제는 시험 대비 개념 퀴즈 중심으로 구성합니다.
- `index.html`에는 중간고사/기말고사 카드만 노출합니다.

## 범위

- 중간고사: 1~7강, 178문제 풀에서 랜덤 33문제
- 기말고사: 8~13강, 247문제 풀에서 강별 랜덤 15문제씩 총 90문제

## Local

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 공개 주소

- GitHub Pages: `https://urusung.github.io/inha-internet-programming-quiz/`

## 구성

- `index.html`: 중간고사/기말고사 선택 첫 화면
- `midterm.html`: 중간고사 대비 개념 퀴즈
- `final.html`: 기말고사 대비 개념 퀴즈
- `files/`: 원본 수업 자료 및 참고 파일
