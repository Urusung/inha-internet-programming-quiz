# inha-internet-programming-quiz

인하대 인터넷프로그래밍 중간고사, 기말고사, 코드 문제 대비용 정적 퀴즈 페이지입니다.

## Local

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 공개 주소

- GitHub Pages: `https://urusung.github.io/inha-internet-programming-quiz/`

## 구성

- `index.html`: 퀴즈 선택 첫 화면
- `midterm.html`: 중간고사 대비 문제
- `final.html`: 기말고사 대비 문제
- `code-quiz.html`: 코드 해석 및 빈칸 문제
- `code/`: 코드 예제 파일
- `files/`: 원본 수업 자료 및 참고 파일
